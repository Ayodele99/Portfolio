# Bike Rental Analysis

## Description
This project analyzes bike rental data to identify trends and patterns using Python.

## Objective
To explore how factors such as weather and time of day influence bike rental demand.

## Tools and Technologies
- Python
- Jupyter Notebook
- pandas, matplotlib, seaborn

## How to Run
- Open `bike_rental_final_code.ipynb` in Jupyter Notebook.
- Run all cells to reproduce the analysis.

## Key Insights
- Rentals peak on weekdays during rush hours.
- Weather conditions significantly influence rental demand.
