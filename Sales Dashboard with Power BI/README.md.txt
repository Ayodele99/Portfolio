# Power BI Dashboard

## Description
This project uses Power BI to create a dashboard visualizing key business metrics.

## Objective
To help business teams make data-driven decisions by visualizing key performance indicators.

## Tools and Technologies
- Power BI

## Key Features
- Interactive charts for revenue and customer trends.
- Filtered views for regional performance analysis.

## How to Open
- Download `BI Classwork 3.pbix`.
- Open it using Power BI Desktop.
